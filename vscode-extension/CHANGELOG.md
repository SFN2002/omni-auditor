# Changelog

All notable changes to the "Omni-Auditor" extension will be documented in this file.

## [0.1.0] - 2025-05-25

### Added
- Initial release.
- **Diagnostics**: Security and complexity findings mapped to the VS Code Problems panel with severity-aware icons.
- **Hover Information**: Risk scores and entropy metrics shown on hover for Python symbols.
- **Risk Dashboard**: Webview panel with interactive charts and file-level risk summaries.
- **Status Bar**: Real-time composite risk score with color coding (green / amber / red).
- **Auto-analyze on Save**: Optional automatic analysis when saving Python files.
- **Commands**: Analyze Current File, Show Risk Dashboard.
